All

Option Possible Values Simplified description
bubbles true
false 		Turn event bubbling on or off
Example
{% highlight javascript %} var options = { bubbles: false }; var tapRecognizer = new Tap(domElement, options); var pointerListener = new PointerListener(domElement, { supportedGestures : [tapRecognizer] }); {% endhighlight %}

Tap

Option Possible Values Simplified description
maxDuration any number If the pointer is longer on the surface than the given number (in milliseconds), Tap becomes invalid.
maxDistance any number If the pointer is moved more than the given distance in px, Tap becomes invalid.
Example
{% highlight javascript %} var options = { maxDuration: 200, maxDistance: 30 }; var tapRecognizer = new Tap(domElement, options); var pointerListener = new PointerListener(domElement, { supportedGestures : [tapRecognizer] }); {% endhighlight %}

Press

Option Possible Values Simplified description
minDuration any number If the pointer is removed from the surface before the given number in milliseconds, Press becomes invalid.
maxDistance any number If the pointer is moved more than the given distance in px, Press becomes invalid.

Pan

Option Possible Values Simplified description
supportedDirections DIRECTION_ALL, DIRECTION_HORIZONTAL, DIRECTION_VERTICAL

or a list[] with one or more of the following:
DIRECTION_LEFT, DIRECTION_RIGHT, DIRECTION_UP, DIRECTION_DOWN 		Define the directions which Pan is valid for.
Example
{% highlight javascript %} var panOptions = { supportedDirections : DIRECTION_HORIZONTAL }; var panRecognizer = new Pan(domElement, panOptions); var pointerListener = new PointerListener(domElement, { supportedGestures : [panRecognizer] }); {% endhighlight %}

Block

By default, contactjs recognizes several gestures simultaneously. You can prevent this using .block.
{% highlight javascript %} var twoFingerPan = new TwoFingerPan(domElement); var pinch = new Pinch(domElement); twoFingerPan.block(pinch); pinch.block(twoFingerPan); var pointerListener = new PointerListener(domElement, { supportedGestures : [twoFingerPan, pinch] }); {% endhighlight %}
Now, pinch will not be recognized during a twoFingerPan and vice versa.
